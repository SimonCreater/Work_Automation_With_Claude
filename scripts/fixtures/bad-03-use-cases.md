# 유스케이스 (Use Cases) — 일부러 깨뜨린 음성 테스트 픽스처

> 이 파일은 R3 검증기가 결함 있는 산출물을 실제로 차단하는지 보여주기 위한
> **고의로 잘못 만든 샘플**이다. 유스케이스가 1건뿐이고 예외 흐름이 없다.
> `scripts/selftest.sh` 의 음성 테스트가 이 파일로 검증기를 호출하며,
> 검증기는 반드시 exit code 2(차단)로 응답해야 한다.

---

## UC-1: 종목 리서치 메모 초안 생성

- **ID**: UC-1
- **Primary Actor**: 애널리스트
- **Trigger**: 애널리스트가 종목명을 입력한다

### Normal Flow (정상 흐름)
1. 애널리스트가 종목명을 입력한다
2. 시스템이 자료를 수집해 메모 초안을 만든다
3. 애널리스트가 초안을 저장한다
